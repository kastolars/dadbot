# Dadbot Discord Bot

## Installation
Option 1: Run `npm install` at project root. 

Option 2 (Docker): run `docker build -t dadbot:latest .` to build the image.

## Usage
Option 1: Run `npm start`.

Option 2 (Docker): Run `docker run -d --name dadbot dadbot:latest`
